Contenido Programático

Comprender la arquitectura de ArcGIS y preparar el entorno de trabajo híbrido (Desktop-Cloud).
  • Introducción a la Ciencia de los Datos Geográficos: Conceptos clave de geodesia, sistemas de coordenadas (WGS84 vs Locales) y tipos de datos.
  • Interfaz de ArcGIS Pro: Manejo de cintas de opciones, paneles de contenido, catálogo y configuración de proyectos (.aprx).
  • Conectividad con ArcGIS Online: Inicio de sesión, licenciamiento y el concepto de "Organización".
  • Datos Online: acceso a datos en línea desde la Biblioteca Esri en ArcGIS PRO, Conexiones STAC.
Crear y administrar bases de datos espaciales.
  • Geodatabase: Creación de Geodatabases, Reglas de atributos, Dominios y Subtipos para integridad de datos.
  • Creación de datos Vectoriales: Creación de puntos líneas y polígonos, edición de elementos, Tabla de Atributos.
  • Edición Avanzada: Herramientas de digitalización, topología de geodatabase para corregir errores de superposición y ajustes de precisión.
  • Importar y exportar Datos: Importación de Excel, KML y Shapefiles hacia la Geodatabase u otras rutas.
  • Tarea - Aplicación de lo aprendido
Resolver problemas territoriales mediante herramientas de análisis vectorial y ráster.
  • Análisis Vectorial: Herramientas de proximidad (Buffer), superposición (Intersect, Union) y extracción.
  • ModelBuilder: Automatización de flujos de trabajo mediante modelos visuales sin necesidad de programación.
  • Tarea - Aplicación de lo aprendido
Comunicar información compleja a través de mapas estéticos y técnicamente correctos.
  • Simbologías: Tipos, edición de simbología, Clasificación por valores únicos.
  • Etiquetado Dinámico: Expresiones Arcade para etiquetas conjuntas, Convertir etiquetas.
  • Estilos: Crear y administrar estilos nuevos para estandarizar simbologías (STYLX).
  • Diseño de Layouts: Diseños predeterminados, Creación de series de mapas, Leyendas y textos dinámicos, Exportación en serie, Exportación de archivo de diseño (.pagx).
  • Tarea - Aplicación de lo aprendido
Publicar contenido y configurar aplicaciones web interactivas.
  • Publicación de Capas: Compartir Web Layer y Web Map desde ArcGIS Pro hacia la nube.
  • Configuración de Map Viewer: Ventanas emergentes (Pop-ups) optimizadas y simbología inteligente (Smart Mapping).
  • Actualización de capas online: Edición de datos y leyendas de capas alojadas en ArcGIS Pro.
  • Field Maps: Diseño de formularios de atributos para la edición de datos en campo.
Desplegar soluciones para la toma de datos en tiempo real y monitoreo.
  • ArcGIS Survey123: Diseño de formularios para recolección de datos en campo, Informes PDF automáticos.
  • ArcGIS Dashboards: Creación de tableros con indicadores clave (KPIs), gráficos dinámicos y mapas en tiempo real.
  • Actualización de capas online: Edición de datos y leyendas de capas alojadas en ArcGIS Pro.
  • ArcGIS StoryMaps: Narrativa geoespacial para presentar informes técnicos de forma atractiva.
